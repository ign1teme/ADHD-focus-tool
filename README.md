# ADHD专注计时器 - Electron桌面应用

## 功能特点

- ✅ 全局置顶窗口，始终可见
- ✅ 主窗口：完整功能界面
- ✅ 悬浮窗：紧凑的计时器窗口
- ✅ 无边框透明悬浮窗
- ✅ 跨应用程序使用

## 安装步骤

### 1. 安装 Node.js

如果还没有安装 Node.js，请从 https://nodejs.org/ 下载并安装。

### 2. 安装依赖

在项目目录下打开命令行，运行：

```bash
npm install
```

这会安装 Electron 和其他必要的依赖包。

## 运行应用

### 开发模式运行

```bash
npm start
```

这会启动应用的开发版本。

## 打包应用

### Windows 打包

```bash
npm run build:win
```

打包后的文件在 `dist` 目录下。

### Mac 打包

```bash
npm run build:mac
```

### Linux 打包

```bash
npm run build:linux
```

## 使用说明

### 主窗口

- 启动应用后会打开主窗口
- 包含完整的番茄钟、任务列表、统计等功能
- 窗口始终置顶，不会被其他窗口遮挡

### 悬浮窗

- 通过菜单 "窗口 → 悬浮窗" 打开
- 紧凑的计时器界面
- 无边框、半透明
- 可以拖动到屏幕任意位置
- 点击 "展开完整界面" 返回主窗口

## 快捷操作

- 拖动悬浮窗：点击顶部区域拖动
- 关闭悬浮窗：点击右上角红色圆点
- 切换窗口：使用菜单栏

## 注意事项

- 首次运行可能需要授予通知权限
- 数据保存在 localStorage 中
- 关闭所有窗口会退出应用（Mac除外）
